# **App Name**: Hourly Reminder

## Core Features:

- Start/Stop Timer: Allow the user to start and stop the hourly timer.
- Alarm Trigger: Trigger an alarm sound every hour after the timer starts.
- Persistent Timer: Maintain timer state even when the app is in the background.
- Customizable Alarm Sound: Option to select a custom sound for the alarm.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a calming and focused feel.
- Background color: Very light gray (#F5F5F5).
- Accent color: Indigo (#3F51B5), slightly brighter than primary, for start/stop buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, will provide a modern look.
- Simple, clear icons for timer controls (start, stop, settings).
- Clean, uncluttered layout with prominent start/stop button and clear time display.
- Subtle animations when the timer starts and when the alarm goes off.
